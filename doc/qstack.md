# qstack

## スタック構造

### フレーム

フレームは手続きの呼び出しごとに作成されるスタック上の領域です。
フレームには以下の４つの領域があります。
 
- 評価領域<br>
  スタックの先頭に位置する領域で、式の評価を行うために使用します。
- ローカル変数領<br>
  評価領域の次に位置する領域で、ローカル変数を格納します。
- 旧フレームポインタ<br>
  ひとつ前のフレームポインタを格納する領域です。
  現在のフレームポインタはこの位置を指しています。
- 引数領域<br>
  手続きに渡された引数を格納する領域です。
